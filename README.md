# 🌕 HR Pulse - Premium Management Platform

A high-performance Human Resources dashboard and scheduling system designed for **Aurum Studio**. Built with **React**, **TypeScript**, and **Firebase**, this platform provides real-time analytics and automated staff management.

## 🚀 Key Features

### 📊 Dynamic HR Dashboard
- **Real-Time Staff Metrics**: Live tracking of active employees and department distribution.
- **Company Health Engine**: Automated attendance percentages and holiday monitoring.
- **Performance History**: Data-driven charts that update based on staff activity.

### 📅 Automated Smart Calendar
- **Internet Sync**: Automatically fetches Sri Lankan public holidays and Poya days via API.
- **Privacy-First Design**: Users only see their own personal leave requests and public holidays.
- **Auto-Update**: Zero maintenance required—the calendar automatically refreshes for 2026, 2027, and beyond.

### 👤 Profile & Identity
- **High-Fidelity Editor**: Premium modal-based profile management with avatar picker.
- **Security-First**: Contact information updates are protected by robust Firestore security rules.

## 🛠️ Technical Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS (Vite)
- **Backend**: Firebase Auth, Cloud Firestore
- **Icons**: Lucide React
- **Notifications**: Sonner
